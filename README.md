# ha-idotmatrix

A custom Home Assistant integration for iDotMatrix.

The Home Assistant "equivalent" (*not so complete yet*) of [the iDotMatrix mobile application](https://play.google.com/store/apps/details?id=com.tech.idotmatrix).
